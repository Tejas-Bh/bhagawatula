About

To get started, in the menu, click "Tickets"

Here, there will be a list of all of the tickets, and a create button.

To create a ticket, click the "create" button.

To update/edit a ticket, click the pencil icon beside the ticket ID.

To add a new user, log into the web server.

Change the directory into Aht-Flask-Tickets, by running the command: cd Aht-Flask-Tickets/

Initialize the virtual environment by doing: source env/bin/activate

Finally, run the add_user script: python3 scripts.py add_user

This script will walk you through adding a new user.
